# 공유 캘린더 (Share Calendar)

여러 사용자가 일정을 함께 보고 관리할 수 있는 캘린더 프로그램입니다.
트리뷰로 개인/공용 캘린더를 나눠서 관리하고, 공용 캘린더는
그룹 단위로 MySQL에 저장해 멤버끼리 공유합니다.

5명 팀 프로젝트로 진행했고, 저는 팀장을 맡아 공용 캘린더 쪽을 담당했습니다.

## 사용 기술
C# WinForms, MySQL, Git / Sourcetree

## 제가 맡은 부분
- 개인/공용을 나누는 트리뷰 구조 설계
- 로그인하면 개인 캘린더 노드가 자동으로 생성되도록 처리
- 공유 캘린더를 만드는 폼과 노드 생성
- 공유 캘린더 정보를 share_group 테이블에 저장해서 그룹 단위로 관리
- 권한에 따른 삭제/탈퇴 처리
  - 공유 캘린더 삭제는 만든 사람만 가능
  - 일반 멤버는 본인만 그룹에서 나갈 수 있음
  - 만든 사람이 삭제하면 참여 중이던 멤버들 노드에서도 같이 사라지게 연동

## 실행할 때
Google Calendar 연동에 `client_secrets.json`이 필요합니다.
보안 때문에 repo에는 올리지 않았고, Google Cloud Console에서 받아서 넣으면 됩니다.
